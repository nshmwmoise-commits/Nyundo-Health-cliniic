          CURRENCY EXCHANGE WEBSITE 
A Currency Exchange Website is a web application that allows users to convert money from one currency to another.
It helps users know the value of one currency in relation to another, for example converting US Dollars (USD) to Rwandan Francs (RWF), or Euros (EUR) to British Pounds (GBP).

            HOW IT WORKS 
  The user enters:

the amount of money
the currency they want to convert from
the currency they want to convert to

Then the system calculates the converted value using exchange rates and displays the result instantly. 
          FEATURES OF THE SYSTEM
  Currency selection (from and to currencies)
Input field for amount
Real-time or fixed exchange rate calculation
Display of converted results
Input validation to prevent errors
